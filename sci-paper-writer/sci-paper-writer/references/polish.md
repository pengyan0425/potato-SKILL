# Manuscript Polish & Review Checklist

## 1. Consistency Check

### Internal Consistency
- [ ] Results section numbers match Discussion interpretations
- [ ] Abstract numbers match Results section
- [ ] All figures/tables referenced in text exist
- [ ] Abbreviations defined on first use, used consistently
- [ ] Terminology is consistent throughout (don't switch between synonyms for key terms)
- [ ] N (sample size) is consistent across sections

### Narrative Consistency
- [ ] Introduction's gap → Methods' approach → Results' findings → Discussion's interpretation form a coherent arc
- [ ] The research question stated in Introduction is answered in Conclusion
- [ ] Claims in Discussion are supported by Results

## 2. Flow & Logic

### Paragraph Level
- [ ] Each paragraph has a clear topic sentence
- [ ] Sentences within a paragraph relate to each other
- [ ] Logical transitions between sentences

### Section Level
- [ ] Sections flow logically into each other
- [ ] Transition sentences between sections where needed
- [ ] No abrupt topic changes

### Document Level
- [ ] Funnel structure in Introduction (broad → specific)
- [ ] Methods ordered to match the flow of Results
- [ ] Discussion moves from findings → interpretation → limitations → future

## 3. Language & Style

### Academic Tone
- [ ] No colloquialisms or slang
- [ ] Appropriate hedging (not over-hedged or under-hedged)
- [ ] Objective tone in Results (no "excitingly", "surprisingly")
- [ ] Professional but readable

### Clarity
- [ ] No sentences > 35 words (break them up)
- [ ] No jargon without definition
- [ ] Acronyms spelled out on first use
- [ ] Each sentence has one clear meaning

### Grammar
- [ ] Subject-verb agreement
- [ ] Correct tense usage (past for methods/results, present for established facts)
- [ ] Parallel structure in lists
- [ ] No dangling modifiers

## 4. Format & Requirements

### Journal Compliance
- [ ] Abstract within word limit
- [ ] Correct section structure for target journal
- [ ] Figure/table format meets requirements
- [ ] Reference format matches journal style
- [ ] Cover letter prepared (if required)

### Word Count
- [ ] Total within journal limit
- [ ] No section disproportionately long or short

## 5. Figures & Tables

### Figures
- [ ] All axes labeled with units
- [ ] Legends are self-explanatory
- [ ] Font size readable at publication size
- [ ] Color-blind friendly palette
- [ ] Resolution ≥ 300 DPI

### Tables
- [ ] Clear column headers
- [ ] Units specified
- [ ] Footnotes explain abbreviations/symbols
- [ ] Not duplicating information in figures

## 6. References

- [ ] All in-text citations have corresponding reference entries
- [ ] All reference entries are cited in text
- [ ] Format consistent (APA, Vancouver, etc. per journal)
- [ ] DOIs included where available
- [ ] Recent references included (not all > 10 years old)
